Learn Plover!

These drills are from the Learn Plover! Book, for use with Plover, which is part of the Open Steno Project. For any lesson, add #random to the end of the web address to show the words in a random order instead of the normal order, which is usually alphabetical, unless it has a more meaningful order like a story. For any lesson, add #repeat to the end of the web address to show all the words in the lesson three times, in a random order.
